TubeLog プライバシーポリシー

最終更新日: 2026年6月5日

日本語 English

本ポリシーは、Chrome 拡張機能「TubeLog」（以下「本拡張」）の利用に伴い取り扱う情報について説明するものです。 本拡張は YouTube および Google LLC の公式製品ではありません。

1. 運営者

本拡張の開発・提供: TubeLog 開発者
お問い合わせ: GitHub Issues

2. 収集・保存する情報の概要

本拡張は、広告配信や行動分析のためのトラッキングを行いません（Firebase 等の分析連携は使用しません）。

データの種類保存場所送信先（第三者）
マイリスト（フォルダ・動画・メモ等） 端末内（chrome.storage.local 等） なし（ユーザー操作時を除く）
チャンネル一覧・UI 設定 端末内 なし
YouTube Data API キー（任意入力） 端末内 ユーザー操作時に Google（googleapis.com）のみ
Google アカウント情報（メールアドレス等） 端末内 Google ログイン時に Google OAuth のみ
バックアップ JSON ユーザーの Google ドライブ（アプリケーションデータ領域） ユーザーがバックアップ実行時に Google のみ
ライセンスキー・認証結果 端末内 ユーザーが認証実行時に Polar（api.polar.sh）のみ

3. 利用目的

  • マイリスト・チャンネル一覧の管理、表示、同期
  • ユーザー設定の保存・復元
  • ユーザーが任意で設定した YouTube Data API による動画・再生リスト情報の取得
  • ユーザーが任意で有効化した Google ドライブへのバックアップ・復元
  • ユーザーが入力した有料ライセンス（Polar）の検証
  • YouTube 上での拡張 UI の提供（ユーザーが YouTube を閲覧しているページ上）

3.1 非公式・非提携の表示

TubeLog は YouTube および Google LLC の公式製品・提携サービスではありません。 YouTube は Google LLC の商標です。

3.2 YouTube 利用規約への同意

本拡張の利用者は、 YouTube 利用規約 および YouTube API サービス利用規約 を遵守する必要があります。 本拡張は、規約で禁止される自動スクレイピングやロボットによるアクセスは行いません。 YouTube Data API を利用する場合は、利用者自身の Google Cloud プロジェクトに適用される条件（クォータ等）にも従います。

4. 第三者サービス

4.1 Google

  • Google ドライブ（appdata）: ユーザーがログインしバックアップを実行した場合のみ、当該 Google アカウントのドライブ内アプリデータ領域に JSON を保存します。
  • Google OAuth: ドライブ連携時に drive.appdata スコープのみを要求します。認証後、設定画面のログイン表示用に Google Drive API（about.get）からメールアドレスを取得し、端末内に保存することがあります。
  • YouTube Data API v3: ユーザーが API キーを設定した場合のみ、キーを用いて Google の API を呼び出します。クォータと利用条件はユーザーの Google Cloud プロジェクトに適用されます。キーは端末内に保存され、運営者のサーバーには送信しません。JSON エクスポート・ドライブバックアップへのキー含有は、同期設定でユーザーが明示的にオンにし同意した場合のみです。

4.2 Polar

有料機能（プレミアムプラン）の販売・請求は Polar 経由で行われ、Google および YouTube による販売・請求ではありません。 支払い・返金に関するお問い合わせは Polar の購入ページおよび本ポリシー記載の運営者へご連絡ください。

有料機能を利用する場合、ユーザーが入力したライセンスキーを HTTPS で Polar の API に送信し、有効性を検証します。 検証結果は端末内にキャッシュします（再検証の間隔は最大 24 時間）。

4.3 個人情報の販売

本拡張は、ユーザーの個人情報を販売しません。

5. Google API から受け取った情報の利用（Limited Use）

本拡張が Google API から受け取った情報の利用および転送は、 Google API Services User Data Policy に加え、限定使用要件（Limited Use）を含む Chrome Web Store ユーザーデータに関するポリシー に従います。

6. データの保存期間・削除

  • 端末内データは、ユーザーが拡張のデータを削除する、拡張をアンインストールする、またはブラウザの拡張データを消去するまで保持されます。
  • Google ドライブ上のバックアップは、ユーザーがドライブ上で削除するまで保持されます。
  • Google ドライブ連携のログアウトは、設定画面から実行できます。

7. セキュリティ

通信は可能な限り HTTPS を使用します。API キー・ライセンスキー・バックアップデータは端末内またはユーザーが選択した Google アカウント配下に保存され、 運営者による中央サーバーへの一括収集は行いません。

8. ポリシーの変更

本ポリシーは必要に応じて更新します。重要な変更がある場合は、本ページの更新日を変更します。

9. お問い合わせ

本ポリシーに関するお問い合わせは GitHub Issues までご連絡ください。

TubeLog Privacy Policy

Last updated: June 5, 2026

This policy describes how the Chrome extension TubeLog (the “Extension”) handles information. The Extension is not an official product of YouTube or Google LLC.

1. Operator

Developer: TubeLog developer
Contact: GitHub Issues

2. Data we process

The Extension does not use analytics or advertising trackers.

DataStorageThird parties
Playlists, folders, notes, channel lists, UI settings On your device (chrome.storage.local, etc.) None (unless you initiate an action below)
YouTube Data API key (optional) On your device Google (googleapis.com) when you use API features
Google account info (email address) On your device Google OAuth when you sign in for Drive backup
Backup JSON Your Google Drive app data folder Google when you run backup
License key and validation result On your device Polar (api.polar.sh) when you validate a license

3. Purposes

  • Manage and display your playlists and channel lists
  • Store and restore settings
  • Fetch YouTube metadata when you provide an API key
  • Backup and restore to Google Drive when you enable it
  • Validate premium licenses via Polar when you enter a key
  • Provide Extension UI on YouTube pages you visit

3.1 Unofficial status

TubeLog is not an official or affiliated product of YouTube or Google LLC. YouTube is a trademark of Google LLC.

3.2 YouTube terms of use

By using the Extension, you agree to comply with the YouTube Terms of Service and the YouTube API Services Terms of Service. The Extension does not perform prohibited automated scraping or robot access. If you use the YouTube Data API, you must also comply with the terms of your Google Cloud project (including quota).

4. Third-party services

We do not sell your personal information.

Google: Drive app data backup (drive.appdata scope only), email for login display via Drive about.get, and optional YouTube Data API calls using your key (stored on your device only; not sent to our servers). Your API key is included in export/backup files only if you enable that option in Sync settings and confirm. Quota applies to your Google Cloud project.
Polar: Premium plans are sold and billed through Polar, not by Google or YouTube. Contact Polar checkout or the operator in this policy for payment and refund inquiries. License validation over HTTPS when you submit a license key.

5. Google API Limited Use

The Extension’s use and transfer of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements, and the Chrome Web Store User Data Policy.

6. Retention and deletion

Local data remains until you clear extension data or uninstall the Extension. Drive backups remain until you delete them in Google Drive. You can sign out of Google Drive backup in the Extension settings.

7. Changes

We may update this policy; the “Last updated” date above will change when we do.
